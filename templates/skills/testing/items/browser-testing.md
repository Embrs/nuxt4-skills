# 瀏覽器自動化測試

## 使用 browser_subagent

透過 Antigravity 的 browser_subagent 工具進行瀏覽器自動化測試。

### 基本用法

```typescript
browser_subagent({
  TaskName: "測試登入流程",
  Task: `
    1. 開啟 http://localhost:3000/sign-in
    2. 在 email 輸入框填入 <測試帳號 email>
    3. 在 password 輸入框填入 <測試帳號密碼>
    4. 點擊登入按鈕
    5. 等待頁面跳轉
    6. 確認跳轉到儀表板頁面
    7. 返回測試結果：是否登入成功
  `,
  RecordingName: "login_test"
});
```

### 常見操作

| 操作 | 說明 |
|------|------|
| 導航 | 開啟指定 URL |
| 點擊 | 點擊按鈕或連結 |
| 輸入 | 在輸入框填寫文字 |
| 選擇 | 選擇下拉選項 |
| 等待 | 等待元素出現或頁面載入 |
| 截圖 | 擷取當前畫面 |
| 驗證 | 確認元素存在或文字內容 |

## 測試案例

### 登入測試
```
Task: 測試登入功能
1. 開啟登入頁面
2. 填寫正確帳密
3. 點擊登入
4. 驗證跳轉到首頁
```

### 新增資料測試
```
Task: 測試新增客戶功能
1. 登入系統
2. 導航到客戶管理頁面
3. 點擊新增按鈕
4. 填寫表單資料
5. 點擊確認
6. 驗證列表中出現新資料
```

### 表單驗證測試
```
Task: 測試表單驗證
1. 開啟新增表單
2. 不填寫任何資料直接送出
3. 驗證必填欄位顯示錯誤提示
```

## 注意事項

1. **明確描述操作**：Task 描述要清晰具體
2. **處理等待**：頁面載入需要時間
3. **錯誤處理**：考慮操作失敗的情況
4. **結果驗證**：明確指定預期結果
