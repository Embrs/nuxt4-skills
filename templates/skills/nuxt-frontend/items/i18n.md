# 國際化 (i18n) 使用

## 支援語言

- 繁體中文（zh_tw）- **預設**
- 英文（en）
- 日文（ja）

## 基本用法

```typescript
// 使用 useI18n
const { t, locale } = useI18n();

// 翻譯
t('common.save')     // 儲存
t('common.cancel')   // 取消
t('common.confirm')  // 確認
```

## 語言檔案位置

```
locales/
├── zh_tw.json    # 繁體中文
├── en.json       # 英文
└── ja.json       # 日文
```

## 新增翻譯

```json
// locales/zh_tw.json
{
  "common": {
    "save": "儲存",
    "cancel": "取消"
  },
  "user": {
    "name": "使用者名稱",
    "email": "電子郵件"
  }
}
```

## 使用限制

### ⚠️ 目前限制
1. **僅用於固定文字**：如按鈕標籤、表格標題
2. **不用於動態內容**：API 返回的資料不進行前端翻譯
3. **後端訊息優先**：錯誤訊息由後端提供三語言

### 正確用法
```pug
//- 固定文字使用 i18n
ElButton {{ t('common.save') }}

//- API 返回的訊息直接顯示
ElMessage(:message="res.status.message[locale]")
```

### 錯誤用法
```pug
//- ❌ 不要翻譯動態資料
span {{ t(user.statusText) }}

//- ✅ 直接顯示後端返回的文字
span {{ user.statusText }}
```

## 切換語言

```typescript
// 切換語言
const { locale } = useI18n();
locale.value = 'en';  // 切換到英文

// 持久化（已自動處理）
// 語言選擇會儲存到 localStorage
```
